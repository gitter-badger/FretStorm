FretStorm
============

This is a Frets On Fire clone, written in C++11.


Installation
-----
On Linux with Autotools.

	$ git clone https://github.com/FretStorm/FretStorm && cd FretStorm
	$ autoreconf -vi && ./configure && make

Execution
----
	$ ./src/fretstorm - 
    OR
    $ ./src/fretstorm /the/path/to/a/file.mid
